# 🍕 Pizza Sales Dashboard

## 📊 Overview
This project presents an interactive **Pizza Sales Dashboard** built using Microsoft Excel.  
The dashboard provides insights into sales performance, customer preferences, and revenue trends.

---

## 🎯 Objectives

- Analyze overall pizza sales performance  
- Identify top-selling pizzas and categories  
- Understand sales trends over time  
- Extract business insights for decision-making  

---

## 🛠️ Tools Used

- Microsoft Excel  
- Pivot Tables and Slicer
- Charts & Visualizations 
---

## 📈 Key Insights

- 🍕 Top-selling pizza categories identified  
- 📊 Revenue trends analyzed across time  
- 🏆 Best-performing products highlighted  
- 📉 Underperforming items observed

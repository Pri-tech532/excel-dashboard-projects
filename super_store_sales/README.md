# 📊 Global Superstore Sales Dashboard

Welcome to my **Global Superstore Sales Dashboard** project!  
This is a **practice project** where I explored the dataset and built a dashboard using **Microsoft Excel**.

---

## 💡 About This Project

- 📊 Created a sales dashboard using **Pivot Tables and Slicers**
- 📈 Visualized data using **Charts and Graphs**
- 🔍 Explored sales, profit, and category trends
- 🎯 Built purely for **learning and practising Excel**

---

## 🛠️ Tools & Skills Used

- Microsoft Excel
- Pivot Tables
- Slicers
- Charts & Graphs (Bar, Pie, Line)

---

## 📂 Dataset Used

- **Source:** Global Superstore 2018
- **Records:** 51,290 orders
- **Coverage:** 165 countries across 5 global markets
- **Categories:** Furniture, Office Supplies, Technology
